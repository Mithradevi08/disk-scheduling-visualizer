# Disk Scheduling Visualizer

## Description
This project is a web-based Disk Scheduling Visualizer that demonstrates how OS disk scheduling algorithms work.

## Algorithms Used
- FCFS
- SSTF
- SCAN
- C-SCAN
- LOOK
- C-LOOK

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Open index.html in browser  
2. Enter disk requests  
3. Select algorithm  
4. View visualization

## Objective
To understand and compare disk scheduling algorithms visually.
